Lógica Computacional

Fluxogramas

Fluxogramas são formas gráficas de representar algoritmos, onde cada operação é representada por uma forma. A principal simbologia usada neles é:

Oval

Demarca o início e o fim do algoritmos

Retângulo

Representa uma operação a ser realizada; algo a ser feito.

Diamante

Indica uma tomada de decisão. Pode representar um se-senão, escolha-caso, etc.

Seta

Mostra para qual direção as informações seguem para chegar nas outras formas.

Algoritmos

Algoritmos são conjuntos de instruções bem definidas para a realização de alguma tarefa (Como as instruções que escrevemos para a máquina ao programar). Podem ser usados em qualquer lugar que precise da automatização de algum processo desde que possa ser descrito de forma clara.

Portugol

LO Portugol é uma pseudo-linguagem (muito parecida com C) com sintaxe em portugues. Por exemplo: “printf()” em C vira “escreva()” em portugol. O Portugol Studio é a IDE usada para escrever, ler e executar códigos em Portugol

Variáveis

São "containers" para armazenar valores que podem ser alterados durante a execução.

var variavel = 0;
variavel = 1 + variavel;

// Alteração da variável 'variavel'

Após ser declarada inicialmente, o valor da variável pode ser modificado conforme o código avança.

Constantes

São "containers" para armazenar valores que não podem ser alterados após a declaração.

const constante = 10;
const resultado = constante * 5;

// 'constante' é usada na conta sem alterações.

Diferente da variável, a constante é usada como base para cálculos, mas seu valor nunca pode ser modificado depois de definida.

Tipos de dados do Portugol

Na programação, os tipos de dados são importantes para definir quais tipos de informações serão manipuladas. Cada dado tem características específicas e é utilizado de acordo com o tipo de operação que será realizada. Abaixo estão alguns dos tipos de dados mais comuns em algoritmos, incluindo o Portugol:

Inteiro

Números inteiros sem parte decimal (ex: 1, 2, 3).

Caracter

Armazena apenas um único caractere (ex: a).

Booleano

Representa um valor verdadeiro ou falso (ex: true, false).

Real

Números com parte decimal (ex: 1.2, 3.14, 5.6).

Cadeia

Uma sequência de caracteres. Também chamada de 'String'. (ex: Isso pode ser uma cadeia).

Vetor

Um conjunto de outros tipos de dados, organizados por índice (ex: vetor[3] = [‘nome’, 3, true]).

Exemplos de algoritmos com Vetores

Vetores são estruturas de dados que armazenam uma sequência de elementos, organizados por índice. Cada elemento pode ser acessado individualmente através do seu índice, facilitando o armazenamento e a manipulação de grandes volumes de dados de forma ordenada. Os vetores são amplamente utilizados em programação para representar listas de itens ou coleções de dados que precisam ser processadas de maneira eficiente.
