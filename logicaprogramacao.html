Lógica de Programação

Línguagens de Programação

Uma linguagem de programação é uma série de instruções que permite o programador a se comunicar com a máquina. Elas podem ser de mais alto ou mais baixo nível, com as de mais alto nível sendo mais verbosas e entendíveis - já as de mais baixo nível, como o Assembly, se aproximam mais do que o computador entende: binário. Assim como as línguas humanas, cada uma delas possuem uma sintaxe e uma semântica próprias e são usadas para criar softwares, como aplicativos, jogos e sistemas web.

C++

Uma linguagem com alto desempenho, amplamente usada no desenvolvimento de sistemas, jogos e softwares que precisam de performance avançada.

JavaScript

Usada principalmente no desenvolvimento web para criar páginas interativas e dinâmicas, sendo a linguagem principal para o front-end e também muito usada no back-end com Node.js.

Java

Uma linguagem robusta e de plataforma independente, conhecida por sua segurança e portabilidade, amplamente utilizada em sistemas corporativos, aplicações móveis e servidores.

Python

Uma linguagem de programação simples e versátil, popular para automação, análise de dados e inteligência artificial e desenvolvimento web.

PHP

Usada principalmente no desenvolvimento web, especialmente para criar sites dinâmicos e interativos, sendo a base de plataformas como WordPress.

Linguagens Interpretadas

Uma linguagem interpretada vai ser executada linha a linha pelo interpretador - sem converter em código de máquina antes. (ex: Python, JavaScript, Ruby)

Linguagens Compiladas

As linguagens compiladas, antes de serem executadas, precisam ser transformadas em código de máquina pelo compilador.

O que é uma IDE?

Uma IDE é o ambiente onde o programador vai escrever os seus programas. Geralmente contam com auxilios como completar automaticamente nomes de arquivos, variaveis, funções, etc.

Ex. ao lado: Visual Studio Code

HTML

O HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo da web. Ele define a disposição e organização dos elementos em uma página, como cabeçalhos, parágrafos, imagens e links. O HTML cria a estrutura básica para o conteúdo.

CSS

O CSS (Cascading Style Sheets) é a linguagem responsável por dar estilo ao conteúdo definido pelo HTML. Com o CSS, é possível aplicar cores, fontes, tamanhos, margens e outros estilos aos elementos da página. Enquanto o HTML organiza o conteúdo, o CSS o torna visualmente atraente e fácil de navegar.

1. Array Crescente e Decrescente

Ordem

Decrescente
Crescente

Array Original:

Clique em "Criar Array" para gerar números aleatórios

Array Ordenado:

Clique em "Ordenar" após criar o array

Ordena um array de 10 números aleatórios entre 1 e 20 em ordem crescente ou decrescente usando o método sort().

Ver Código

 
let switchCrescente1 = document.getElementById('switchCrescente1')
let btnCriarArray1 = document.getElementById('btnCriarArray1')
let btnOrdenar1 = document.getElementById('btnOrdenar1')
let resArrOrdenado1 = document.getElementById('resArrOrdenado1')
let resArrNumeros1 = document.getElementById('resArrNumeros1')

let arrNumeros1 = []

function gerarArrayAleatorio1() {
    let arr = []
    for (let i = 0; i < 10; i++) {
        arr.push(Math.floor(Math.random() * 20) + 1)
    }
    return arr
}

btnCriarArray1.addEventListener('click', () => {
    arrNumeros1 = gerarArrayAleatorio1()
    resArrNumeros1.innerText = arrNumeros1.join(', ')
    resArrOrdenado1.innerText = 'Clique em "Ordenar" após criar o array'
})

btnOrdenar1.addEventListener('click', () => {
    if (!arrNumeros1.length) {
        resArrOrdenado1.innerText = 'Primeiro crie um array.'
        return
    }

    let arrOrdenado1 = arrNumeros1.sort((a, b) => a - b)

    if (!switchCrescente1.checked) {
        arrOrdenado1.reverse()
    }

    resArrOrdenado1.innerText = arrOrdenado1.join(', ')
})

2. Ordenar com Bubble Sort

Array Original:

Clique em "Criar Array" para gerar números aleatórios

Array Ordenado:

Clique em "Ordenar" após criar o array

Implementa o algoritmo Bubble Sort, que compara pares de elementos adjacentes e os troca se estiverem na ordem errada, repetindo até o array estar ordenado.

Ver Código

 
let btnCriarArray2 = document.getElementById('btnCriarArray2')
let btnOrdenar2 = document.getElementById('btnOrdenar2')
let resArrNumeros2 = document.getElementById('resArrNumeros2')
let resArrOrdenado2 = document.getElementById('resArrOrdenado2')
let arrNumeros2 = []

function gerarArrayAleatorio2() {
    let arr = []
    for (let i = 0; i < 10; i++) {
        arr.push(Math.floor(Math.random() * 20) + 1)
    }
    return arr
}

function bubbleSort2(arr) {
    for (let i = 0; i < arr.length; i++) {
        for (let j = 0; j < arr.length - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                let temp = arr[j]
                arr[j] = arr[j + 1]
                arr[j + 1] = temp
            }
        }
    }
    return arr
}

btnCriarArray2.addEventListener('click', () => {
    arrNumeros2 = gerarArrayAleatorio2()
    resArrNumeros2.innerText = arrNumeros2.join(', ')
    resArrOrdenado2.innerText = 'Clique em "Ordenar" após criar o array'
})

btnOrdenar2.addEventListener('click', () => {
    if (!arrNumeros2.length) {
        resArrOrdenado2.innerText = 'Primeiro crie um array.'
        return
    }
    let arrOrdenado = bubbleSort2(arrNumeros2)
    resArrOrdenado2.innerText = arrOrdenado.join(', ')
})

4. Matriz 3x3

Resultado:

Clique em "Gerar Matriz" para preencher os valores

Cria uma matriz bidimensional 3x3 preenchida com números aleatórios entre 1 e 20, usando loops aninhados para percorrer linhas e colunas.

Ver Código

 
let btnGerarMatriz4 = document.getElementById('btnGerarMatriz4')
let resMatriz4 = document.getElementById('resMatriz4')

function gerarNumeroAleatorio() {
    return Math.floor(Math.random() * 20) + 1
}

function gerarMatriz4() {
    let matriz = []
    for (let i = 0; i < 3; i++) {
        let linha = []
        for (let j = 0; j < 3; j++) {
            linha.push(gerarNumeroAleatorio())
        }
        matriz.push(linha)
    }
    return matriz
}

btnGerarMatriz4.addEventListener('click', () => {
    let matriz = gerarMatriz4()

    resMatriz4.innerHTML = `
        

            

                ${matriz[0][0]}
                ${matriz[0][1]}
                ${matriz[0][2]}
            

            

                ${matriz[1][0]}
                ${matriz[1][1]}
                ${matriz[1][2]}
            

            

                ${matriz[2][0]}
                ${matriz[2][1]}
                ${matriz[2][2]}
            

        

    `
})

5. Conversor Fahrenheit/Celsius

Fahrenheit
Celsius

Resultado:

Escolha o sentido e clique em converter

Converte temperaturas entre Fahrenheit e Celsius usando as fórmulas: °C = (°F - 32) × 5/9 e °F = (°C × 9/5) + 32.

Ver Código

 
let inputTemp5 = document.getElementById('inputTemp5')
let switchConversao5 = document.getElementById('switchConversao5')
let btnConverter5 = document.getElementById('btnConverter5')
let resConversao5 = document.getElementById('resConversao5')

function paraCelsius(valor) {
    return ((valor - 32) * 5) / 9
}

function paraFahrenheit(valor) {
    return (valor * 9) / 5 + 32
}

btnConverter5.addEventListener('click', () => {
    let valor = Number(inputTemp5.value)
    if (isNaN(valor)) {
        resConversao5.innerText = 'Digite uma temperatura válida.'
        return
    }

    if (switchConversao5.checked) {
        let resultado = paraFahrenheit(valor).toFixed(1)
        resConversao5.innerText = valor + '°C = ' + resultado + '°F'
    } else {
        let resultado = paraCelsius(valor).toFixed(1)
        resConversao5.innerText = valor + '°F = ' + resultado + '°C'
    }
})

7. Cálculo de IMC

Resultado:

Informe seu peso e altura para calcular

Calcula o Índice de Massa Corporal usando a fórmula: IMC = peso (kg) / altura (m)², útil para avaliar o estado nutricional.

Ver Código

 
let inputPeso7 = document.getElementById('inputPeso7')
let inputAltura7 = document.getElementById('inputAltura7')
let btnCalcular7 = document.getElementById('btnCalcular7')
let resIMC7 = document.getElementById('resIMC7')

function calcularIMC7(peso, altura) {
    if (!altura) {
        return 0
    }
    return (peso / (altura * altura)).toFixed(2)
}

btnCalcular7.addEventListener('click', () => {
    let peso = Number(inputPeso7.value)
    let altura = Number(inputAltura7.value)

    if (peso <= 0 || altura <= 0) {
        resIMC7.innerText = 'Informe peso e altura válidos.'
        return
    }

    let imc = calcularIMC7(peso, altura)
    resIMC7.innerText = 'IMC: ' + imc
})
